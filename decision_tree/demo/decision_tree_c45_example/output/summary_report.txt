C4.5决策树 (Decision Tree C4.5) 实验报告
============================================================

## 数据集要求:
1. X: 特征矩阵，形状为 (n_samples, n_features)
2. y: 标签向量，支持多分类
3. 支持连续和离散特征的混合
4. 自动处理特征类型
5. 适用于多分类问题

## 算法特点:
1. 使用信息增益比作为分裂准则
2. 支持连续和离散特征
3. 内置剪枝功能（PEP剪枝）
4. 处理过拟合问题
5. 相比ID3算法更稳定

## 推荐数据集:
1. sklearn.datasets.make_classification() - 生成分类数据
2. sklearn.datasets.load_iris() - 鸢尾花数据集
3. sklearn.datasets.load_wine() - 葡萄酒数据集
4. sklearn.datasets.load_breast_cancer() - 乳腺癌数据集
5. 任何多分类数据集

## 生成的图片说明:
- 散点图: 显示数据分布和分类结果
- 决策边界图: 显示C4.5决策树的分类边界
- 准确性图: 显示预测正确和错误的点
- 剪枝对比图: 比较剪枝前后的效果

## 与其他决策树算法的比较:
- ID3: 只支持离散特征，使用信息增益
- C4.5: 支持连续特征，使用信息增益比，有剪枝
- CART: 二叉树，支持回归，使用基尼系数
